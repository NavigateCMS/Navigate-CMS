I will use following terms: node (dir), leaf (item).

Managing nodes / leafs:

To Add, Edit, Delete node / leaf you have to select a node / leaf first. This can be done by clicking node / leaf.

Edit, Delete node / leaf - will be performed upon selected node / leaf

Add:
- if node is selected the new item (node or leaf) will be appended to the end of this node as its item
- if leaf is selected the new item will be appended after this leaf

Drag & drop:

You can drag & drop nodes / leafs by dragging the text of a node / leaf
